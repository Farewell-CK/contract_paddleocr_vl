# 服务合同

**合同编号**: CN-003

| 项目        | 说明 |
| ----------- | ---- |
| 甲方        | 北京星河科技有限公司 |
| 乙方        | 上海明远数字有限公司 |
| 合同金额    | 人民币 325,000 元 |
| 签署日期    | 2021年5月18日 |
| 生效日期    | 2025年6月17日 |
| 到期日期    | 2020年12月25日 |

## 第一条 服务内容

乙方应按照甲方要求提供专业化服务，并确保交付成果满足双方约定的质量标准。

## 第二条 费用及支付方式

甲方应在合同生效后五个工作日内支付合同总金额的 50%，剩余款项于项目验收后七日内付清。

## 第三条 保密义务

双方对在合同履行过程中获知的任何商业信息负有严格的保密义务。
